End Assignment - Graphics Programming 2

Trailer

End assignment for the course "Graphics Programming 2" in my second year at Digital Arts & Entertainment. We were given a start of a game engine, where we had to add the required graphics implementations, as well as the needed gameplay functionality for the chosen game.

Implementations made for this project are:

  • Shadow Mapping
  • Sprite Rendering
  • Text Rendering
  • SkyBox
  • Post Processing
  • Particles
  • Skinned Model Animations
  • Character Controller
  • FMOD Sound System
  • Rig Rail Camera System
  • State Pattern
  • Invoker System
  • ...
